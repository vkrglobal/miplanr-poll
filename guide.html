miPlanr Events & Polls v4.0 Guide

  1. Backup your existing GitHub folder.
  2. Copy all v4 files into your miplanr-poll folder.
  3. Run supabase/schema.sql in Supabase SQL Editor.
  4. Check Netlify variables: SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY, RESEND_API_KEY.
  5. Commit to main and Push origin.
  6. Wait for Netlify Published.
  7. Create a fresh event poll from /index.html.
  8. Test email invite, WhatsApp share, one vote only, edit vote, and calendar location.
