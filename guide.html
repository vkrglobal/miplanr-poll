miPlanr Poll v3.0 Guide

Create polls, send secure invite links, prevent duplicate votes, allow vote edits before deadline, include location in calendar links, and notify the organiser when quorum is reached.

Google Sites integration

In Google Sites, use Insert → Embed → By URL and paste https://miplanr.com/index.html. You can also add a button on vkrglobal.net linking to the same URL.
