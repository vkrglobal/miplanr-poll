Dummies guide

  1. Back up your current miplanr-poll folder.
  2. Copy all files from this ZIP into the GitHub miplanr-poll folder and replace files.
  3. Open Supabase → SQL Editor → New query.
  4. Paste and run supabase/schema.sql.
  5. Commit all changes in GitHub Desktop.
  6. Push origin.
  7. Wait for Netlify Published.
  8. Create a brand-new poll at /index.html.
  9. Add date/time fields if known. If left blank, miPlanr applies safe defaults.
  10. Test WhatsApp, email, voting, quorum celebration and calendar links.
