miPlanr Poll v6 Install Guide

  1. Backup existing GitHub folder.
  2. Copy all v6 files into the miplanr-poll folder.
  3. Run supabase/schema.sql in Supabase SQL Editor.
  4. Check Netlify variables.
  5. Commit and push in GitHub Desktop.
  6. Wait for Netlify Published.
  7. Test create poll, invite, vote once, edit vote, calendar location.
