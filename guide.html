Setup checklist
  1. Copy all files into your GitHub miplanr-poll folder.
  2. Run supabase/schema.sql in Supabase SQL Editor.
  3. Check Netlify environment variables: SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY, RESEND_API_KEY.
  4. Commit all changes in GitHub Desktop.
  5. Push origin.
  6. Wait for Netlify to publish.
  7. Open Create Poll.
  8. Create a brand-new poll and use the success panel to share by WhatsApp/email.
