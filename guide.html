miPlanr Poll v6.4 install guide

  1. Backup your current project folder.
  2. Copy all v6.4 files into your GitHub folder.
  3. Run supabase/schema.sql in Supabase SQL Editor.
  4. Commit and push in GitHub Desktop.
  5. Wait for Netlify to publish.
  6. Create a fresh test poll.
