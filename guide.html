Dummies guide

1. Copy all v4.1 files into your GitHub miplanr-poll folder.

2. Run supabase/schema.sql in Supabase SQL Editor.

3. Confirm Netlify has SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY and RESEND_API_KEY.

4. Optional: add OPENAI_API_KEY for translation.

5. Commit all files in GitHub Desktop and Push origin.

6. Wait for Netlify to publish.

7. Create a fresh poll and test invite links, vote editing, calendar location, preview and translate.
