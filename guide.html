Dummies guide — miPlanr Poll 2.0 + 2.1

What this version does

2.0 is focused on the polling workflow: create, save, share, email, vote, and show real results. 2.1 adds integration-ready buttons for calendar, education, CRM and sports platforms.

Setup checklist

  1. Run supabase/schema.sql in Supabase SQL Editor.
  2. Add Netlify environment variables: SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY, and RESEND_API_KEY.
  3. Replace your project files with this package.
  4. Commit all changes in GitHub Desktop.
  5. Push origin.
  6. Wait for Netlify to publish.
  7. Create a new poll at /index.html.
  8. Test the share link, WhatsApp, email invite and voting.

Important

WhatsApp currently means one-click sharing, not automatic WhatsApp Business API sending. The 2.1 integration buttons record requests/interest and are ready for future direct API sync.
