miPlanr Poll v2 — Dummies Guide

Important: This v2 uses Supabase as a database. Create Supabase first, then add the keys to Netlify.

1. Create Supabase project

  1. Go to Supabase and create a new project called miplanr-poll.
  2. Open SQL Editor.
  3. Create a new query.
  4. Open supabase/schema.sql from this project.
  5. Copy all of it, paste into Supabase, click Run.

2. Get Supabase keys

  1. In Supabase go to Project Settings → API.
  2. Copy Project URL.
  3. Copy service_role key. Keep this secret.

3. Add keys to Netlify

  1. Open Netlify project.
  2. Go to Project configuration → Environment variables.
  3. Add SUPABASE_URL.
  4. Add SUPABASE_SERVICE_ROLE_KEY.
  5. Optional later: add RESEND_API_KEY for email invites.

4. Copy files into GitHub folder

Replace your project files with this package. Keep backups first.

C:\Data\Professional\VKR\Tech Projects\miPlanr\GitHub\miplanr-poll

5. Commit and deploy

  1. Open GitHub Desktop.
  2. Summary: Add miPlanr Poll v2 full project
  3. Click Commit to main.
  4. Click Push origin.
  5. Wait for Netlify to publish.

6. Test

  1. Open Create Poll.
  2. Create a poll.
  3. Copy the poll link.
  4. Open the poll link.
  5. Vote.
  6. Refresh the page and check the vote is still there.
  7. Click WhatsApp share.

WhatsApp note: This version supports one-click WhatsApp sharing. Automatic WhatsApp sending requires the WhatsApp Business API later.
