Dummies guide: miPlanr Poll v5.0

  1. Download and extract the ZIP.
  2. Backup your current miplanr-poll folder.
  3. Copy all v5 files into your GitHub folder and replace existing files.
  4. Open Supabase → SQL Editor → New Query.
  5. Open supabase/schema.sql, copy everything, paste into Supabase, click Run.
  6. Confirm Netlify has SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY, RESEND_API_KEY.
  7. Optional: add OPENAI_API_KEY for translation.
  8. Commit to main and push origin in GitHub Desktop.
  9. Wait for Netlify Published.
  10. Create a brand-new poll and test emails, WhatsApp, voting and calendar.
