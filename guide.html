miPlanr Poll v8.0.3 Guide

What changed

This version consolidates matching polls. If you create/share another poll with the same event title, question, poll type and identical options/date-times, responses are stored against the same poll database and will appear in the same results view, subject to the administrator's results visibility setting.

Quorum rule

For calendar polling, quorum is counted per date/time option. If Sarah selects Monday and Tuesday, that is one availability vote for Monday and one availability vote for Tuesday. It is not two votes towards Monday, and it is not two respondents.

Required Supabase step

  1. Open Supabase.
  2. Go to SQL Editor.
  3. Paste and run the contents of supabase/schema.sql.
  4. Deploy the full folder to Netlify.
